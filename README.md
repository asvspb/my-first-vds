# Настройка Ubuntu 24.04 для разработки

Версия: **3.0** (Полностью на Python 🐍)

Универсальный оркестратор для быстрой инициализации и управления VDS на Ubuntu 24.04. Проект полностью переписан на Python с использованием интерактивного TUI (Text User Interface).

## Архитектура v3.0

Проект успешно завершил миграцию с коллекции bash-скриптов на **единый Python-CLI оркестратор**. Все старые `.sh` файлы удалены.

**Ключевые фичи:**
- Единая команда `vds` с графическим интерактивным меню (на базе `questionary`).
- Безопасное управление: пароли скрыты, запросы интерактивны, используется модульная архитектура.
- Структура пакета: `core/`, `zerotier/`, `wireguard/`, `sysinfo/`, `system/`.
- Использование современных библиотек: `typer` (CLI), `requests` (API), `pydantic` (валидация конфигураций), `rich` (красивый вывод), `psutil` (системные метрики).
- Systemd-таймеры для фонового авто-восстановления и синхронизации (watchdog & reconcile).

## Быстрый старт

У вас есть два способа установки.

### Способ 1: С локального компьютера (Рекомендуемый)
Если вы только что купили сервер и у вас есть только его IP и пароль, скачайте и запустите этот скрипт прямо на **своем ноутбуке/ПК**:

```bash
curl -fsSL https://raw.githubusercontent.com/asvspb/my-first-vds/main/local-init.sh | bash
```

Скрипт спросит IP сервера, скопирует на него ваш SSH-ключ (спросит пароль 1 раз), сам установит всю систему и автоматически откроет меню `vds`!

### Способ 2: Изнутри сервера
Если вы уже находитесь на сервере, выполните одну команду:

```bash
curl -fsSL https://raw.githubusercontent.com/asvspb/my-first-vds/main/install.sh | sudo bash
```

*(По желанию, перед этим можно выполнить предварительную базовую подготовку:* `curl -fsSL https://raw.githubusercontent.com/asvspb/my-first-vds/main/preinstall.sh | sudo bash`*)*

После установки просто введите команду `vds` в терминале. Откроется главное интерактивное меню!

```bash
vds                           # Открыть графическое TUI меню
```

Вы также можете вызывать функции напрямую, минуя графическое меню:

```bash
vds --help                    # Справка по всем командам
vds sysinfo                   # Статус сервера (CPU, RAM, Docker, Disk)
vds cleanup                   # Системная очистка
vds server-setup              # Запустить мастер базовой настройки сервера
vds zerotier install          # Установить ZeroTier + ZTNET Panel
vds wireguard install         # Установить WireGuard VPN
```

## Главное меню `vds`

При запуске команды `vds` без аргументов доступен полный арсенал управления:

- **Системный статус (Dashboard)**: Наглядный вывод метрик сервера (на базе `psutil`).
- **Управление ZeroTier**: Полный набор инструментов (Установка, Диагностика, Добавление сетей, Синхронизация топологии `reconcile`, Восстановление NAT, Запуск Watchdog, Полное удаление).
- **Управление WireGuard**: Установка VPN-сервера, добавление новых клиентов с генерацией конфигов, безопасное удаление.
- **Базовая настройка сервера (Setup)**: Интерактивный мастер настройки VDS (Создание пользователя, SSH-ключи, защита SSH, настройка UFW, выделение Swap, установка Docker).
- **Очистка системы (Cleanup)**: Освобождение места на диске (чистка логов, кэшей apt, старых контейнеров Docker, journalctl).

## Systemd-сервисы (ZeroTier)

После установки ZeroTier автоматически настраиваются системные таймеры для поддержания инфраструктуры:

| Сервис | Описание | Интервал |
|--------|----------|----------|
| `vds-watchdog.timer` | Мониторинг и авто-восстановление туннелей ZeroTier | каждые 2 мин |
| `vds-reconcile.timer` | Декларативная синхронизация состояния с `topology.json` | каждые 5 мин |

## Требования

- Ubuntu 24.04 (LTS)
- Права суперпользователя (sudo)
- Доступ к интернету

*(Python 3.10+ и все зависимости устанавливаются автоматически скриптом `install.sh`)*

## Разработка

Для добавления новых команд в проект:
1. Клонируйте репозиторий.
2. Создайте новый модуль в `src/<category>/<module>.py`.
3. Зарегистрируйте команду в `src/main.py` через декораторы `@app.command()` или `@<category>_app.command()`.
4. При необходимости добавьте пункт в графическое меню `main_menu()`.
